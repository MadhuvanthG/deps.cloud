deps.cloud works by crawling well known sources of repositories such as GitLab, GitHub, and BitBucket. A custom integration can be added to support any JSON API.
From the repositories, we're able to infer relationships between repositories by parsing dependency management files. The relationships between systems are then stored and indexed in an easy to query knowledge graph.
